## 1. 디자인 철학

### 1.1 핵심 원칙

| 원칙 | 설명 |
|------|------|
| **다크 우선 (Dark First)** | 기본 테마는 다크. 콘텐츠 집중도 향상, 눈 피로도 감소 |
| **로컬 우선 강조** | "Cloud Sync Disabled" 배지 등 로컬 처리 상태를 UI에 명시 |
| **단순함 (Simplicity)** | 불필요한 장식 요소 없음. 기능 중심의 미니멀 UI |
| **전문성 (Professional)** | Slate 계열 색조 + Inter 폰트로 전문적인 도구 느낌 |
| **접근성 (Accessibility)** | 충분한 색상 대비, 시맨틱 HTML 구조 |

### 1.2 시각적 계층 구조

```
배경 (#111921)
  └─ 사이드바 / 패널 (#1A242F, rgba 오버레이)
      └─ 카드 / 컴포넌트 (#2D3A49)
          └─ 텍스트 (#F8FAFC, #94A3B8)
              └─ 강조 (Primary #1773CF)
```

## 4. 아이콘

- **출처**: 커스텀 SVG (Figma에서 IMAGE-SVG 타입으로 정의)
- **크기**: 주로 12px × 12px ~ 16px × 16px
- **색상**: 컨텍스트 색상 상속 (currentColor 방식 권장)
- **사용 라이브러리**: Lucide React (shadcn/ui 기본 아이콘 셋)

## 8. 테마 전환 구조

CSS 변수 기반으로 라이트/다크 테마 전환을 지원한다:

```css
/* globals.css */
:root {
  /* 다크 테마 (기본) */
  --background: 212 32% 10%;
  --surface: 210 30% 15%;
  --border: 215 25% 17%;
  --primary: 211 79% 45%;
  --foreground: 210 40% 98%;
  --muted-foreground: 215 16% 63%;
}

.light {
  /* 라이트 테마 (선택적) */
  --background: 0 0% 100%;
  --surface: 210 40% 98%;
  --border: 214 32% 91%;
  --foreground: 222 47% 11%;
  --muted-foreground: 215 16% 47%;
}
```

## 9. AdSense 광고 배치 원칙

- **에디터 화면**: 사이드바 하단 (편집 영역과 완전히 분리)
- **PDF 화면**: 사이드바 하단 + 메인 콘텐츠 하단
- **OCR 화면**: 콘텐츠 하단
- **규칙**: 편집 영역(에디터 캔버스, PDF 뷰어)에 오버레이 금지
- **모바일 (375px 미만)**: 광고 숨김 (`hidden sm:block`)
