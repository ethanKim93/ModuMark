## 1. 도메인 개요

### 1.1 도메인 목적

Monetization 도메인은 ModuMark의 **지속 가능한 무료 서비스 운영을 위한 수익 기반**을 담당한다. Google AdSense를 유일한 수익원으로 하여, 사용자 경험을 훼손하지 않는 범위 내에서 광고를 통한 수익을 창출한다.

AdSense 승인은 완료되었으나, **"저가치 콘텐츠(Low-value content)" 정책 위반**으로 광고 게재가 중지될 리스크가 존재한다. 이를 해소하기 위해 공개 페이지 콘텐츠 품질 강화, 네비게이션 일관성 확보, 가이드 콘텐츠 허브 구축이 필요하다.

### 1.2 연결된 비즈니스 문제

| 전체 BRD 문제 ID | 문제 | Monetization 도메인 연관성 |
|-----------------|------|--------------------------|
| - | 1인 개발 체제의 지속 운영 비용 | 광고 수익으로 서버·인증서 비용 충당, 지속 운영 가능성 확보 |
| P2 | 품질 있는 도구가 유료 | **무료 유지를 위한 수익 모델**: AdSense 수익이 무료 서비스 운영의 재정 기반 |
| - | AdSense 저가치 콘텐츠 정책 위반 리스크 | 공개 페이지 콘텐츠 품질 미달 시 광고 게재 중지 → 수익 단절 |

## 5. 성공 지표 (KPI)

| KPI | 정의 | 목표 |
|-----|------|------|
| AdSense 월 수익 | Google AdSense를 통한 월간 광고 수익 | $200 (출시 6개월) |
| RPM (Revenue Per Mille) | 페이지뷰 1,000회당 수익 | $1.5 이상 유지 |
| 광고 뷰어빌리티 | 화면에 실제 표시된 광고 비율 (IAB 기준) | 50% 이상 |
| 광고 CTR | 광고 노출 대비 클릭 비율 | 0.5% 이상 |
| CLS 증가 | 광고 로딩으로 인한 레이아웃 이동 | CLS 0.1 이하 유지 |
| 광고로 인한 이탈율 변화 | 광고 도입 전후 페이지 이탈율 차이 | 5% 이내 |
| Lighthouse SEO 점수 | Google Lighthouse SEO 카테고리 점수 | 90점 이상 (모든 공개 페이지) |
| 공개 페이지 콘텐츠 충분성 | SSR 콘텐츠 800자 이상 보유 페이지 수 | 전체 공개 페이지 100% |
| 가이드 콘텐츠 페이지 수 | /guide/* 하위 가이드 페이지 수 | 5개 이상 |

| PDF 하위 라우트 SSR 복원 | /pdf/merge, /pdf/split, /pdf/ocr — 각 페이지 고유 SSR 콘텐츠 |
| 개인정보처리방침 GDPR 보강 | GDPR 옵트아웃 조항, 쿠키 정책 상세화 |

| sitemap 업데이트 | /about, /guide/*, /pdf/merge, /pdf/split, /pdf/ocr 추가 |
